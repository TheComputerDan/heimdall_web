Heimdall

Introduction

Description:

A centralized tool for managing containers and vms across multiple hosts.

Why?

Both small businesses and enterprise environments can and do suffer from the need to manage multiple endpoints for containers, vms, and other services of the sort. The goal of this project to create a tool that allows people to easily manage different endpoints centrally.

Technologies:

  • Golang
  • Docker
  • VMWare (Planned)

Endpoints

Docker:

Leveraging the Docker API to report and manage Docker hosts.

VMWare:

We hope to integrate Fusion, Workstation, and ESXi. With intent to include vCenter for more advanced management.
